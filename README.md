## Versões necessárias

Versão NodeJS: 22.9.0

## O que foi reforçado neste projeto?

- HTML;
- CSS;
- ReactJs;
- React Hooks;
- React Router;
- Firebase Firestore.

## Como rodar?

### `npm install`

Instala as dependências do projeto criando a pasta node_modules.

### `npm start`

Roda o projeto na porta 3000.
